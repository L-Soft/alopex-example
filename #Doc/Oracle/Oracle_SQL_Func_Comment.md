### 1. 각 테이블 코멘트 조회 하기

~~~sql
SELECT 
    ALL_T.TABLE_NAME
    ALL_TC.COMMENTS
FROM 
    ALL_TABLES ALL_T,
    ALL_TAB_COMMENTS ALL_TC
WHERE 
    ALL_T.TABLE_NAME = ALL_TC.TABLE_NAME
ORDER BY ALL_T.TABLE_NAME;
~~~

2. 테이블의 컬럼 정보 조회 하기

~~~sql
SELECT
    A.OWNER,
    A.TABLE_NAME,
    A.COLUMN_ID,
    A.COMMENTS,
    A.DATA_TYPE,
    A.DATA_LENGTH,
    A.NULLBLE
FROM
    ALL_TAB_COLUMNS A,
    ALL_COL_COMMENTS B
WHERE
    A.TABLE_NAME = B.TABLE_NAME AND
    A.COLUMN_NAME = B.COLUMN_NAME
    A.COLUMN_NAME = "찾는 컬럼명"
ORDER BY A.TABLE_NAME, A.COLUMN_ID
~~~
